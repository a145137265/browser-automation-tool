# Browser Automation Suite - 浏览器自动化控制工具

[English](#english) | [中文](#中文)

---

## 中文

专业的浏览器自动化控制工具，支持多实例管理。

### 功能
- 多浏览器支持 (Chrome/Firefox/Edge)
- 浏览器指纹定制
- Cookie/Session管理
- 批量操作
- 性能监控
- 异常自动恢复

### 安装
```bash
# Node.js版本
npm install

# Python版本
pip install -r requirements.txt
```

### 使用
```bash
# Node.js
node src/index.js

# Python
python src/main.py
```
